
Domain Member Link Module

Description
------------
This module creates a link on users' profile page to each of their assigned domains.


Installation
-------------
1. Place the entire module directory into your Drupal sites/all/modules/
   directory.

2. Enable the module by navigating to:
     administer > modules

3. This module creates 2 permissions 'view own assigned domains' and
    'view all users assigned domains' which can be assigned from the 'User management >> Permissions' page.
